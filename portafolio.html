SISTEMATIZACION DE HORAS SOCIALES (APLICACION WEB)

Descripción: Este proyecto aborda una problemática del centro educativo relacionada con el registro del servicio social obligatorio para los estudiantes de bachillerato, requisito necesario para obtener el título. las páginas físicas de registro se pierden o dañan, complicando el seguimiento de las horas realizadas de los estudiantes.

Para darle solución a este problema creamos con un grupo de compañero una aplicación web progresiva desarrollada con los lenguajes de programacion: PYTHON, HTML, CSS, MySQL, SQLite y Flask. Este proyecto frece un sistema de gestión de horas de servicio social en tiempo real, accesible desde cualquier lugar. Incluye un inicio de sesión para tres tipos de usuarios: alumnos, maestros y administradores.Y cada usuario de los 3 mencionados juega un rol diferente. Los estudiantes pueden consultar proyectos disponibles, enviar formularios para la validación de horas, y los maestros pueden aprobarlas o validarlas de manera eficiente y alos alumnos automaticamente se le agregan a sus horas sociales realizadas.

ARDUINO-termometro

Descripción: Este proyecto lo desarrolle con el propósito de practicar y aprender a construir un termómetro con Arduino Uno.

Este circuito mide la temperatura exterior del ambiente .La temperatura ambiental se mide con un sensor y se muestra en una pantalla LCD en grados Celsius. El proyecto fue programado en C++ y utiliza varios componentes de Arduino, como un sensor de temperatura, cables jumper, una pantalla LCD y otros elementos.
